# STEGANOGRAPHY

It's simple Steganography encoded/decoded tool for CS232

Clone it using git:
```sh
$ git clone https://github.com/sonlhcsuit/steganography
```
Running:
```sh
$ python3 -m venv ./venv
$ pip install -r requirements.txt
```

```sh
(env)$ python app.py
```

CLI
```shell
$ python3 cli.py -h
```
