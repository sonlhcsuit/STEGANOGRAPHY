# STEGANOGRAPHY

It's simple Steganography encoded/decoded tool for CS232

Clone it using git:
```sh
$ git clone https://github.com/sonlhcsuit/STEGANOGRAPHY.git
```
Running:
```sh
$ python3 -m venv ./env
$ pip install -r requirements.txt
```

```sh
(env)$ python app.py
```
